# Responsive reference adaptation

Preserve ordering, emphasis, grouping, navigation intent, state meaning, and
parent/child geometry. At narrower widths, wrap/reflow/scroll only as needed;
at wider widths, constrain or use explicitly approved expansion. Never remove
sections, move the primary scan action, collapse secondary actions, or swap the
visual hierarchy merely because the reference width differs. Test breakpoints,
long localization, RTL, safe areas, keyboard, and 200% text scale.
